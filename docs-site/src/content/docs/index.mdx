## What is Repotype?

ESLint checks your code. Prettier formats it. **Repotype checks your repository.**

Enforce folder structure, file naming, documentation standards, and schema validation across your entire codebase.

<CardGrid stagger>
  <Card title="Folder Structure" icon="folder">
    Define required directories, allowlists, and path conventions.
  </Card>
  <Card title="File Standards" icon="document">
    Enforce naming conventions, companion files, and content schemas.
  </Card>
  <Card title="Documentation Quality" icon="pencil">
    Validate frontmatter, required sections, and template completion.
  </Card>
  <Card title="CI-Ready" icon="rocket">
    Run in pre-commit hooks and CI pipelines with JSON output.
  </Card>
</CardGrid>

## Quick Start

```bash
# Install
npm install -D @supernal/repotype

# Initialize config
npx repotype init .

# Validate
npx repotype validate .

# Fix auto-fixable issues
npx repotype fix .
```
